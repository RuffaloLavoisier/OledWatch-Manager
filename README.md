## Feature(plan)

1. 손목시계와 연결하기  
2. 시계 현재 시간 설정하기 
3. 손목시계 설정 및 웹 타임 반영  
4. 손목시계 탑재된 RGB 모듈을 제어
5. 손목시계 위치를 음성으로 확인
6. RSSi와 구글맵을 연동하여 위치를 지도에 표시(상호간 거리 알림)
7. 스마트폰 PUSH 알림 전송
