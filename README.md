## Structure

손목시계
1. Push 알림 받아오기(전화,문자,메신저..etc)  
2. 스마트폰과 거리 받아오기(BT)


스마트폰
1. 주변 손목시계와 연결하기  
2. 시계 현재 시간  
3. 손목시계 현재 위치 받아오기(BT)  
4. 손목시계 설정 및 웹 타임 반영
